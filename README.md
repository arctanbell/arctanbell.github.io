# GitHub个人网站

这是一个简洁、现代的个人网站模板，专为GitHub Pages设计。

## 网站特性

- 🎨 响应式设计（适配手机、平板、桌面）
- ⚡ 快速加载
- 🌙 深色/浅色主题切换
- 📱 移动端友好
- 🔗 社交媒体集成
- 📧 联系表单（邮件链接）

## 部署到GitHub Pages

### 方法1：使用你的用户名仓库（推荐）

1. **创建仓库**：在GitHub上创建一个名为 `你的用户名.github.io` 的仓库
   - 例如：如果你的GitHub用户名是 `johnsmith`，仓库名应该是 `johnsmith.github.io`

2. **上传文件**：将这个项目的所有文件上传到该仓库

3. **访问网站**：几分钟后，你的网站将在 `https://你的用户名.github.io` 上线

### 方法2：使用现有仓库的gh-pages分支

1. **在现有仓库中**：将这些文件推送到 `gh-pages` 分支

2. **启用GitHub Pages**：
   - 进入仓库 Settings → Pages
   - 在 Source 部分选择 "Deploy from a branch"
   - 选择 `gh-pages` 分支和 `/root` 文件夹
   - 点击 Save

3. **访问网站**：网站将在 `https://你的用户名.github.io/仓库名` 上线

## 自定义内容

编辑 `index.html` 文件来修改以下内容：

- **个人信息**：姓名、职位、简介
- **头像**：替换 `avatar.jpg` 或更新HTML中的图片链接
- **社交媒体链接**：更新各个社交平台的URL
- **项目展示**：添加或修改项目卡片
- **联系方式**：更新邮箱地址

## 文件结构

```
├── index.html          # 主页面
├── style.css           # 样式文件
├── script.js           # 交互脚本
├── avatar.jpg          # 头像图片（可选）
└── README.md           # 说明文档
```

## 预览效果

网站包含以下部分：
- **导航栏**：带有主题切换按钮
- **英雄区域**：显示你的姓名、职位和简介
- **关于我**：详细介绍和技能
- **项目展示**：你的作品集
- **联系信息**：社交媒体和邮箱链接
- **页脚**：版权信息

## 注意事项

- 所有文件都已优化，可以直接使用
- 如果要添加头像，请将图片命名为 `avatar.jpg` 并放在根目录
- 网站完全静态，无需服务器端代码
- 支持自定义域名（需要额外配置）
