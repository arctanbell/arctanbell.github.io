## 我的项目（基于真实 GitHub 项目）

### YOLOv4 目标检测优化
基于 Darknet 框架的 YOLOv4 实现，针对特定场景进行性能优化和精度调优。支持 Windows 和 Linux 平台，适用于实时目标检测应用。

**技术栈**: C, CUDA, OpenCV  
**GitHub**: [darknet](https://github.com/arctanbell/darknet)

### 视频超分辨率与稳定化
结合 RealBasicVSR 和 Mesh-Flow-Video-Stabilization 技术，实现高质量的视频增强和稳定化处理。适用于监控视频、无人机航拍等场景。

**技术栈**: Python, PyTorch, OpenCV  
**GitHub**: [RealBasicVSR](https://github.com/arctanbell/RealBasicVSR), [Mesh-Flow-Video-Stabilization](https://github.com/arctanbell/Mesh-Flow-Video-Stabilization)

### MediaPipe 应用开发
基于 Google MediaPipe 框架开发的实时姿态估计和手势识别应用，支持跨平台部署。

**技术栈**: C++, Python, MediaPipe  
**GitHub**: [mediapipe](https://github.com/arctanbell/mediapipe)

### VS-Net 视觉定位
实现基于投票机制的视觉定位系统，用于 AR/VR 和机器人导航场景。

**技术栈**: Python, PyTorch, 3D 计算机视觉  
**GitHub**: [VS-Net](https://github.com/arctanbell/VS-Net)
